# FALE-Home_Unified_Lab
Unified Cybersecurity Lab tracking via Jira & GitHub | **Author Node: Kali Linux (T7 Persistence) on Mini PC**

This repository contains the automation scripts and documentation for the FALE Enterprise Supply Chain Simulation.

## Included Scripts
* **watchman.sh**: Real-time directory monitoring for warehouse file changes.
* **update_lab.sh**: Automated system maintenance and package updates for the Kali node.

## Project Management
Task tracking and sprint documentation are managed via Jira.
